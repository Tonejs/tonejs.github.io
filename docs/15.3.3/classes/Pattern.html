    Class Pattern<ValueType>

    Pattern arpeggiates between the given notes in a number of patterns.

    Example

    const pattern = new Tone.Pattern((time, note) => {
    	// the order of the notes passed in depends on the pattern
    }, ["C2", "D4", "E5", "A6"], "upDown");
